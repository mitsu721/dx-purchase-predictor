# DX商材購入予測モデル

このプロジェクトは、企業の属性・財務情報・アンケート結果をもとに、DX商材を購入する可能性を予測する分類モデルです。LightGBMを用いてF1スコアを最適化し、特徴量設計・相互作用・業界分析・SHAP解釈を含みます。

## 特徴
- アンケート逆スコア化による不満度抽出
- 相互作用特徴量の設計
- 財務比率・対数変換による正規化
- 業界別購入率分析と高購入業界フラグ
- SHAPによる特徴量重要度の可視化（未実施、できれば実装）

## 使用技術
- Python / Pandas / LightGBM / SHAP(未使用) / Seaborn / Matplotlib
